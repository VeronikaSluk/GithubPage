ОС GNU/Linux

Ласкаво просимо на сторінку, присвячену операційній системі GNU/Linux. Тут ви дізнаєтеся про базову інформацію про систему, її дистрибутиви, команди та приклад скрипта на BASH.

Коротка інформація про ОС GNU/Linux

Linux — це сімейство Unix-подібних операційних систем, які використовують ядро ​​Linux розробки фінно-американського програміста Лінуса Торвальдса. ОС, що використовують ядро ​​Linux, називаються дистрибутивами Linux, і є такими ж операційними системами як Microsoft Windows або Apple macOS, але з однією дуже важливою особливістю — їх вихідні коди є відкритими, оскільки вони поширюються під ліцензією GNU GPL, яка передбачає створення вільного та відкритого програмного забезпечення (open source software). Це означає, що будь-який користувач має право вивчати та змінювати вихідний код.

Популярні дистрибутиви GNU/Linux

  • Ubuntu — це ідеальний вибір як для професіоналів, так і для початківців, завдяки своїй простоті та функціональності.
  • Debian — стабільний та надійний дистрибутив з великим репозиторієм програмного забезпечення. Базовий дистрибутив для багатьох інших версій Linux.
  • Fedora — швидкий та інноваційний дистрибутив, що часто використовується розробниками через постійні оновлення новітніх технологій.
  • Arch Linux — дистрибутив для досвідчених користувачів, який надає максимальну гнучкість і налаштування системи.
  • CentOS — дистрибутив, орієнтований на серверні рішення, базується на кодовій базі Red Hat Enterprise Linux (RHEL).

Приклади команд для роботи в Linux CLI

pwd

Відображення поточного робочого каталогу.

echo "текст"

Виведення повідомлення або значення змінної на екран.

ls

Перелік файлів і каталогів у поточному каталозі.

cd [каталог]

Переміщення в інший каталог.

touch [файл]

Створення нового порожнього файлу.

mkdir [каталог]

Створення нового каталогу.

cp [файл1] [файл2]

Копіювання файлу з одного місця в інше.

mv [файл1] [файл2]

Переміщення або перейменування файлу.

rm [файл]

Видалення файлу.

history

Відображення історії команд у поточній сесії терміналу.

BASH-скрипт, що імітує авторизацію в системі


#!/bin/bash

# Задання правильних значень логіна і пароля
CORRECT_USERNAME="admin"
CORRECT_PASSWORD="password123"

# Запит логіна у користувача
read -p "Введіть логін: " username

# Запит пароля у користувача
read -s -p "Введіть пароль: " password
echo # Перехід на новий рядок

# Перевірка введених даних
if [[ "$username" == "$CORRECT_USERNAME" && "$password" == "$CORRECT_PASSWORD" ]]; then
    echo "Авторизація успішна. Вітаємо, $username!"
else
    echo "Помилка авторизації. Невірний логін або пароль."
fi

Цей скрипт імітує процес авторизації в системі. Він запитує у користувача логін та пароль, після чого перевіряє правильність введених даних.
